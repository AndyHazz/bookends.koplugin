# Bookends 4.4.0 — release notes

## New: "+ New blank preset" tile

A tile at the end of your preset library creates a fresh, empty preset and drops you straight into the Bookends menu to start filling it in.

## New: rename your current preset

The Preset submenu now has a **Rename (Preset name)…** row — shows which preset you're editing and renames it on tap. No trip to the library needed.

## Changed: "Save as new preset…" is now "Save current as new preset…"

Clearer about what it does: it duplicates your current look into a new preset. Your ongoing edits already autosave.
